# Symbolic Math Interpreter

A symbolic mathematics engine that parses, simplifies, and differentiates expressions.

**Language:** Python or C++  
**Key Concepts:** Parsing (shunting-yard), expression trees, symbolic differentiation, simplification

## Goals

- Tokenize and parse mathematical expressions
- Perform symbolic differentiation and simplification
- REPL interface for interactive use
