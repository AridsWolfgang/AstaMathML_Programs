# Game Physics Engine

A rigid-body physics engine for 2D/3D simulation.

**Language:** C++  
**Key Concepts:** Collision detection, rigid-body dynamics, constraints, integration schemes

## Goals

- Implement AABB, OBB, and SAT collision detection
- Resolve collisions with impulse-based response
- Build a demo with rendering (SDL, OpenGL, or similar)
