# Numeric Methods Engine

A library of numerical algorithms for scientific computing.

**Language:** C or C++  
**Key Concepts:** Root finding, numerical integration, ODE solvers, interpolation, linear system solvers

## Goals

- Implement bisection, Newton-Raphson, Runge-Kutta, etc.
- Benchmark against standard libraries (GSL, LAPACK)
- Provide a clean C API for reuse
