# Image Classifier

A computer vision system for image recognition and classification.

**Language:** Python (prototyping) or C++ (production inference)  
**Key Concepts:** CNNs, transfer learning, data augmentation, PyTorch/TensorFlow

## Goals

- Train a classifier on CIFAR-10, MNIST, or custom dataset
- Implement a CNN from scratch (C++) or use PyTorch (Python)
- Achieve >90% accuracy on a benchmark dataset
