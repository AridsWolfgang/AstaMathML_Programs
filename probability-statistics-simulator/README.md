# Probability & Statistics Simulator

A simulation environment for probability distributions and statistical tests.

**Language:** Python or JavaScript  
**Key Concepts:** Monte Carlo, distributions, hypothesis testing, CLT, Bayesian inference

## Goals

- Simulate coin flips, dice rolls, random walks, etc.
- Visualize PDFs, CDFs, histograms
- Implement A/B testing and Bayesian updates
