## Structure

Each project lives in its own folder with its own README, build system, and dependencies. They are independent — no shared build — to keep things portable and focused.

## Principles

- **Standalone** — each project is self-contained
- **From scratch** where it matters, with libraries where it doesn't
- **Language-appropriate** — the right tool for each problem
