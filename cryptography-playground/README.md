# Cryptography Playground

A collection of cryptographic algorithm implementations and experiments.

**Language:** C or C++  
**Key Concepts:** Symmetric/asymmetric encryption, hashing, digital signatures, elliptic curves

## Goals

- Implement AES, RSA, SHA-256 from scratch
- Demonstrate encryption/decryption workflows
- Explore attacks (frequency analysis, timing attacks)
