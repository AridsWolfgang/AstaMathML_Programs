# Spam / Sentiment Classifier

A text classification system for spam detection and/or sentiment analysis.

**Language:** Python  
**Key Concepts:** Naive Bayes, logistic regression, SVM, embeddings, fine-tuning BERT

## Goals

- Build a spam filter (binary classification)
- Build a sentiment analyzer (positive/negative/neutral)
- Compare classical ML vs. deep learning approaches
