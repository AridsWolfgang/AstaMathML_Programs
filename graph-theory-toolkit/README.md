# Graph Theory Toolkit

A library and CLI for graph algorithms and analysis.

**Language:** C++ or Python  
**Key Concepts:** BFS/DFS, shortest paths, MST, flows, graph coloring, centrality

## Goals

- Implement classic graph algorithms (Dijkstra, Kruskal, Ford-Fulkerson)
- Visualize graphs and their properties
- Solve problems on random and real-world graphs
