# Linear Algebra Visualizer

An interactive tool for visualizing vectors, matrices, and transformations.

**Language:** JavaScript (WebGL/Canvas)  
**Key Concepts:** Vector spaces, linear transformations, eigenvectors, SVD

## Goals

- Render 2D/3D vectors and transformations in the browser
- Interactive rotation, scaling, and projection
- Visualize eigendecomposition and singular value decomposition
