# Custom Math Library

A from-scratch mathematical library in C.

**Language:** C  
**Key Concepts:** Arbitrary precision arithmetic, trigonometry, logarithms, special functions

## Goals

- Implement `sin`, `cos`, `exp`, `log`, `sqrt` without libm
- Support arbitrary-precision integers and floats
- Provide a header-only C library for other projects
